#M5スティックENVIII
M5StickCの温度、湿度、圧力をグラフで表示します。
「M5StickC ENV Hat III」を装着する必要があります。

![Image](/display.jpg)
* [ReadMe.md](/README.md)

＃ 特徴
温度、湿度、気圧を8秒ごとに取得します。
また、更新ごとにデータを表示し、2 分ごとのデータを保持します。
保持したデータはM5Stickを回転させることで折れ線グラフで表示できます。
1分後にスリープ モードに入りますが、データの収集は継続されます。

＃ 使用法
* 電源長押し: 電源を切ります。
* BtnA押下: スリープから復帰します。
* BtnB押下: 手動でスリープに入ります。
* 時計回りに回転: 温度と湿度のグラフを表示します。
* 反時計回りに回転: 圧力グラフを表示します。
